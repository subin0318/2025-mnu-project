# 2025 MNU Project
목포대학교 LMS 포털 기능을 클론하고, LMS의 **마이페이지 내 파일관리 시스템을 재구현**하며  
UI/UX를 리디자인하여 보다 편한 인터페이스와 업그레이드된 기능을 제공하는 것을 목표로 하는  
목포대학교 **융합소프트웨어학과 2학년 팀 프로젝트**입니다.
